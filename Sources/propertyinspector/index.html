The basic formula is:

  • Start the work timer (25 minutes)
  • Work until the timer expires
  • Take a short break (5 minutes)
  • Repeat this process 3 more times, then
  • Take a long break (25 minutes)

This action will audibly and visibly alert you when the timer has expired. The "when the timer ends" option lets you choose how many button presses are needed to turn off the alarm and start the next timer. You can pause the timer at any time with a short press to pause and later resume. Pressing and holding for 2 seconds resets the internal cycle counter back to the first work phase.

Note: This action times things differently than the standard method; the fourth break is one long break interval instead of a short break into a long break.
